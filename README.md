# 📚 Simple Book & Author REST API

Welcome to a humble and straightforward Spring Boot REST API for managing **Books** and **Authors**.  
No fancy stuff here — just clean code, PostgreSQL integration, and a sprinkle of Docker magic.

---

## 🚀 Tech Stack

- **Backend:** Java (JDK 21)
- **Framework:** Spring Boot
- **Database:** PostgreSQL
- **Build Tool:** Maven (of course, who doesn’t love it?)
- **Containerization:** Docker (CLI-friendly)
- **Security:** ❌ Not yet (soon™)
- **API Docs:** ❌ Coming later
